# stm32-gpio-and-timing-fundamentals

This repository compares multiple timing mechanisms on the STM32 MCU, including blocking delays, SysTick-based scheduling, hardware timer interrupts, and PWM output, highlighting tradeoffs in determinism, CPU utilization, and scalability.

## Structure
- `01_blocking_delay/`
- `02_systick_nonblocking/`
- `03_timer_interrupt/`
- `04_pwm_pwm_dimming/`
- `docs/` – detailed technical notes
